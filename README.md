## Descripción
Trabajo práctico donde realicé una página de superhéroes dinámica donde se puede buscar por nombre, agregar superhéroes al catálogo así como también eliminar cualquiera. Implementé lógica de JS como manejo de objetos, manipulación de arreglos, escucha de eventos globales, detección de clicks específicos, captura de strings provenientes de inputs del usuario, etc. El workflow del práctico consistió en moverse a través de distintas ramas durante el proyecto. Trabajé HTML, CSS y Bootstrap en rama main, así como también JavaScript tanto en la rama feature/catalogo como en feature/formulario.

## Funcionalidades

* **Catálogo Dinámico**: Renderizado de cartas desde JavaScript utilizando `innerHTML`.
* **Formulario**: Captura de inputs ingresados por el usuario para agregar contenido.
* **Búsqueda**: Filtrado por nombre según input del usuario.
* **Eliminación**: Botones funcionales para eliminar cards.

## Tecnologías Utilizadas

* **HTML5**: Estructura semántica de la aplicación.
* **CSS y Bootstrap**: Estilos, diseño de tarjetas y efectos de animación.
* **JavaScript (Vanilla)**: Manipulación dinámica del DOM e inyección de contenido.
* **Git & GitHub**: Control de versiones utilizando flujo de trabajo basado en ramas (`main`, `feature/catalogo`, `feature/formulario`).
